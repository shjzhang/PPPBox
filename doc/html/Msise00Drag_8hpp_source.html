152  /*
153  * NOTES ON INPUT VARIABLES:
154  * UT, Local Time, and Longitude are used independently in the
155  * model and are not of equal importance for every situation.
156  * For the most physically realistic calculation these three
157  * variables should be consistent (lst=sec/3600 + g_long/15).
158  * The Equation of Time departures from the above formula
159  * for apparent local time can be included if available but
160  * are of minor importance.
161  *
162  * f107 and f107A values used to generate the model correspond
163  * to the 10.7 cm radio flux at the actual distance of the Earth
164  * from the Sun rather than the radio flux at 1 AU. The following
165  * site provides both classes of values:
166  * ftp://ftp.ngdc.noaa.gov/STP/SOLAR_DATA/SOLAR_RADIO/FLUX/
167  *
168  * f107, f107A, and ap effects are neither large nor well
169  * established below 80 km and these parameters should be set to

187  /*
188  * OUTPUT VARIABLES:
189  * d[0] - HE NUMBER DENSITY(CM-3)
190  * d[1] - O NUMBER DENSITY(CM-3)
191  * d[2] - N2 NUMBER DENSITY(CM-3)
192  * d[3] - O2 NUMBER DENSITY(CM-3)
193  * d[4] - AR NUMBER DENSITY(CM-3)
194  * d[5] - TOTAL MASS DENSITY(GM/CM3) [includes d[8] in td7d]
195  * d[6] - H NUMBER DENSITY(CM-3)
196  * d[7] - N NUMBER DENSITY(CM-3)
197  * d[8] - Anomalous oxygen NUMBER DENSITY(CM-3)
198  * t[0] - EXOSPHERIC TEMPERATURE
199  * t[1] - TEMPERATURE AT ALT
200  *
201  *
202  * O, H, and N are set to zero below 72.5 km
203  *
204  * t[0], Exospheric temperature, is set to global average for
205  * altitudes below 120 km. The 120 km gradient is left at global
206  * average value for altitudes below 72 km.
207  *
208  * d[5], TOTAL MASS DENSITY, is NOT the same for subroutines GTD7
209  * and GTD7D
210  *
211  * SUBROUTINE GTD7 -- d[5] is the sum of the mass densities of the
212  * species labeled by indices 0-4 and 6-7 in output variable d.
213  * This includes He, O, N2, O2, Ar, H, and N but does NOT include
214  * anomalous oxygen (species index 8).
215  *
216  * SUBROUTINE GTD7D -- d[5] is the "effective total mass density
217  * for drag" and is the sum of the mass densities of all species
218  * in this model, INCLUDING anomalous oxygen.
219  */
